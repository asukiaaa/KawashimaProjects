# SonicSensorSoundPlayer
A project to play mp3 file by ultora sonic sensor value.

# Connection
- A1 - SonicSensor1 Trigger
- A2 - SonicSensor1 Echo
- A3 - SonicSensor2 Trigger
- A4 - SonicSensor2 Echo
- D9 - DFPlayer play detect
- D10 - SoftSerial RX
- D11 - SoftSerial TX

# Files on SD
```
mp3
 |- 0001.mp3 // music
 |- 0002.mp3 // stop sound
 |- 0003.mp3 // music
 \- 0004.mp3 // stop sound
```
